## 题目
用"结论先行→分层支撑（MECE）"结构，重写以下模糊的表达：

**原版**："我们的产品增长不错，但这个季度营收有点下滑，可能是市场环境不好，也可能是我们上次涨价导致的。不过用户留存好像还行，只是新用户来得少了。总之情况比较复杂，需要进一步分析。"

### 金字塔重写版

**结论先行**：本季度营收下滑15%，核心原因是新用户获取放缓，而非存量用户流失。

**分层支撑**：

├─ 📉 营收下滑15%（Q2 vs Q1）
│  ├─ 存量用户贡献稳定（留存率92%，持平）
│  ├─ 新用户获取量下降40%（核心问题）
│  └─ 客单价微升5%（涨价生效，但不足以对冲）
│
├─ 🔍 新用户下降的两个可能原因
│  ├─ 外部：市场整体流量下降（行业数据验证，大盘降20%）
│  └─ 内部：提价后转化率降低（涨价前转化率3.2%→涨价后2.1%）
│
└─ ✅ 结论：问题出在"涨价→新用户转化率下降"的传导链，存量业务健康
    ├─ 建议：恢复基础版定价或做差异化定价
    └─ 预期：2个月内新用户恢复到正常水平
