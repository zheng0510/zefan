## 题目
一种罕见病在人群中的患病率是0.1%。某检测方法对患病者的阳性检出率是99%，对健康者的误报率是1%。如果一个人检测结果为阳性，他真正患病的概率是多少？

---

## 直觉回答（错误）

大多数人会回答"99%"——因为检测准确率是99%。

## 贝叶斯推理

**第一步：明确问题**
- 求：P(患病 | 阳性)
- 已知：
  - P(患病) = 0.001（基础概率）
  - P(阳性 | 患病) = 0.99（敏感度）
  - P(阳性 | 健康) = 0.01（误报率）

**第二步：构建贝叶斯公式**
```
P(患病 | 阳性) = P(阳性 | 患病) × P(患病) / P(阳性)
```

**第三步：计算 P(阳性)**
```
P(阳性) = P(阳性 | 患病) × P(患病) + P(阳性 | 健康) × P(健康)
        = 0.99 × 0.001 + 0.01 × 0.999
        = 0.00099 + 0.00999
        = 0.01098
```

**第四步：代入计算**
```
P(患病 | 阳性) = (0.99 × 0.001) / 0.01098
                = 0.00099 / 0.01098
                ≈ 0.0902
                ≈ 9%
```

## 结论

**检测阳性后，真正患病的概率只有约9%，不是99%。**

## 为什么差距这么大？

关键在于**基础概率极低（0.1%）**。

用更直观的方式理解：假设10000人：
- 真患病者：10人，其中9.9人检测阳性
- 健康者：9990人，其中99.9人误报阳性
- 阳性总人数：9.9 + 99.9 = 109.8人
- 真患病占阳性比例：9.9 / 109.8 ≈ 9%

**误报的绝对数量（99.9人）远超真阳性（9.9人）**——因为健康人群基数太大。
