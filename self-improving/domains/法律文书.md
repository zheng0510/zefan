# 法律文书书写优化知识库

## 第一部分：基础规范 (第1-10轮)

### 1.1 法律文书通用格式
- 标题：文书名称 + 当事人 + 案由
- 首部：法院/机构名称、案号
- 正文：事实、理由、判决/意见
- 尾部：日期、签名、印章

### 1.2 合同文书
- 合同标的条款
- 权利义务条款
- 违约责任条款
- 争议解决条款
- 附则附件

### 1.3 诉讼文书
- 起诉状
- 答辩状
- 上诉状
- 申诉状
- 申请书

## 第二部分：专业文书 (第11-20轮)

### 2.1 法律意见书
- 事实概述
- 法律分析
- 结论建议

### 2.2 尽职调查报告
- 公司概况
- 法律风险
- 建议措施

### 2.3 律师函
- 发函事由
- 法律依据
- 对方责任
- 后果告知

## 第三部分：高级文书 (第21-33轮)

### 3.1 公司章程
### 3.2 股东协议
### 3.3 知识产权协议
### 3.4 劳动争议文书
### 3.5 仲裁文书

## 核心原则
- 用词准确、严谨
- 结构清晰、逻辑严密
- 引用法律条文准确
- 格式规范统一
