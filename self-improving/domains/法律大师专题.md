# 法律大师专题 - 学习笔记

## 第601轮：国际商事法庭制度

### 一、国际商事法庭概述

#### 1.1 主要国际商事法庭

```
专门性国际商事法庭：
├── 新加坡国际商事法庭（SICC）
│   ├── 2015年成立
│   ├── 可受理国际商事案件
│   └── 判决可在新加坡执行
│
├── 迪拜国际金融中心法院（DIFC Court）
│   ├── 2004年成立
│   ├── 适用普通法
│   └── 国际化法官组成
│
├── 阿布扎比全球市场法院（ADGM Court）
│   ├── 2015年成立
│   ├── 适用英国普通法
│   └── 国际化商事纠纷解决
│
├── 中国国际商事法庭（CICC）
│   ├── 2018年成立（最高法下设）
│   ├── 受理国际商事案件
│   └── 推动"一带一路"争端解决
│
└── 荷兰国际商事法庭（Netherlands Commercial Court）
    ├── 2019年成立
    ├── 可使用英语审理
    └── 提升荷兰商事争端解决吸引力
```

#### 1.2 CICC特色机制

```
中国国际商事法庭（CICC）创新机制：
├── 专家委员会
│   ├── 国际商事专家委员
│   ├── 提供专业咨询
│   └── 参与调解
│
├── 国际商事调解机构
│   ├── 一站式纠纷解决平台
│   ├── 调解+仲裁+诉讼衔接
│   └── "一带一路"国际商事调解中心
│
├── 判决跨境执行机制
│   ├── 依据双边条约
│   ├── 依据互惠关系
│   └── 推动判决互认
│
└── 诉讼与仲裁衔接
    ├── 仲裁保全协助
    ├── 仲裁司法审查
    └── 裁决执行高效
```

### 二、国际商事法庭优势

#### 2.1 与传统诉讼/仲裁比较

| 维度 | 国际商事法庭 | 国内法院 | 国际仲裁 |
|------|--------------|----------|----------|
| 专业性 | 高（商事专家法官） | 取决于法官背景 | 高（当事人选定专家） |
| 国际化 | 强（国际法官/语言） | 弱 | 强 |
| 执行效率 | 强（判决直接执行） | 强 | 需依据公约（纽约公约） |
| 灵活性 | 一般 | 低 | 高 |
| 保密性 | 弱（公开审理） | 弱 | 强 |
| 费用 | 相对较低 | 低 | 高 |
| 时间 | 中等 | 可能较长 | 中等 |

#### 2.2 选择策略

```
选择国际商事法庭的情形：
├── 需要判决在多个国家执行
├── 争议涉及公共利益或强制性规定
├── 需要司法解释明确法律规则
└── 仲裁可能存在管辖权争议

选择仲裁的情形：
├── 需要高度保密
├── 当事人希望选定专家裁判
├── 需要跨境执行（纽约公约）
└── 不希望公开审理
```

### 三、中国国际商事法庭实务

#### 3.1 管辖范围

```
CICC管辖案件：
├── 第一国际商事法庭（深圳）
│   ├── 广东、广西、海南、湖南
│   └── 标的额≥3亿元
│
├── 第二国际商事法庭（西安）
│   ├── 其他省份
│   └── 标的额≥3亿元
│
└── 第三国际商事法庭（北京）（拟设）
    ├── 北京及周边
    └── 标的额≥3亿元
```

#### 3.2 审理特点

```
CICC审理特色：
├── 审判庭组成
│   ├── 审判长（最高法法官）
│   └── 国际商事专家委员（参与调解/咨询）
│
├── 程序特点
│   ├── 可使用英文提交材料
│   ├── 国际商事调解优先
│   └── 在线诉讼便利
│
└── 判决执行
    ├── 中国境内：直接执行
    └── 境外：依据条约/互惠关系
```

### 四、实务要点

```
国际商事法庭运用策略：
1. 评估管辖连接点
2. 比较诉讼/仲裁优劣
3. 关注判决跨境执行机制
4. 利用一站式纠纷解决平台
5. 专家委员会资源运用
```

## 第602轮：跨境破产与债务重组

### 一、跨境破产法律框架

#### 1.1 主要跨境破产模式

```
跨境破产两种主要模式：
├── 普及主义（Universality）
│   ├── 一个主要破产程序
│   ├── 全球资产纳入程序
│   └── 全球债权人公平受偿
│
└── 地域主义（Territoriality）
    ├── 各国分别进行破产程序
    ├── 各自处理境内资产
    └── 程序之间协调困难
```

#### 1.2 联合国跨境破产示范法（UNCITRAL Model Law）

```
示范法核心机制：
├── 承认外国程序
│   ├── 主要程序（Center of Main Interests, COMI）
│   └── 非主要程序
│
├── 救济措施
│   ├── 自动中止（Stay）
│   ├── 债务暂停偿付
│   └── 资产保全
│
└── 合作机制
    ├── 法院间合作
    ├── 破产管理人合作
    └── 信息交换
```

### 二、中国跨境破产实践

#### 2.1 法律依据

```
中国跨境破产法律依据：
├── 《企业破产法》第五条
│   ├── 承认和执行外国破产判决/裁定
│   └── 以条约为基础，或互惠关系
│
├── 最高人民法院指导意见
│   ├── 跨境破产案件审理指引
│   └── 互惠关系认定标准
│
└── 实践发展
    ├── 早期较为保守
    ├── 近期逐步开放
    └── 首例承认外国破产程序（2019，香港）
```

#### 2.2 典型案例分析

```
案例：美国太阳能公司破产案
├── 美国法院启动破产程序
├── 该公司在华有子公司和资产
├── 美国破产管理人请求中国法院承认程序
├── 中国法院依据互惠原则予以承认
└── 实现跨境资产协调分配
```

### 三、债务重组跨境协调

#### 3.1 跨境债务重组类型

```
债务重组模式：
├── 法庭外重组（Workout）
│   ├── 债权人与债务人协商
│   ├── 不需要法院介入
│   └── 灵活性高
│
├── 破产重整（Court-supervised Reorganization）
│   ├── 法院监督下重整
│   ├── 强制批准机制
│   └── 对异议债权人有约束力
│
└── 预打包破产（Pre-packaged Bankruptcy）
    ├── 事前达成重组协议
    ├── 快速通过法院批准
    └── 提高效率
```

#### 3.2 跨境协调难点

```
跨境债务重组难点：
├── 法律冲突
│   ├── 破产原因认定不同
│   ├── 重整计划批准标准不同
│   └── 债权人优先权顺序不同
│
├── 资产追回
│   ├── 资产跨国转移
│   ├── 欺诈性转让
│   └── 资产追回程序复杂
│
└── 债权人协调
    ├── 多国债权人
    ├── 不同法律背景
    └── 协调成本高
```

### 四、实务要点

```
跨境破产实务策略：
1. 提前规划资产分布
2. 了解主要司法管辖区破产法
3. 利用UNCITRAL Model Law
4. 协调多国程序
5. 保护债权人利益
6. 关注中国实践发展
```
