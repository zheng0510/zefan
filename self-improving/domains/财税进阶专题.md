
# 财税进阶专题 - 学习笔记

## 第511轮：跨境电商税务专题

### 一、跨境电商税务模式

#### 1.1 跨境电商模式类型

```
跨境电商模式：
├── 9610：跨境电子商务零售直接出口
│   └── 适用于B2C小包出口
│
├── 1210：保税跨境贸易电子商务
│   └── 适用于备货模式
│
├── 9710：跨境电商B2B直接出口
│   └── 适用于B2B大货
│
└── 9810：跨境电商出口海外仓
    └── 适用于海外仓模式
```

### 二、出口退税要点

```
出口退税条件：
├── 已报关出口离境
├── 已收汇
├── 已取得增值税专用发票
└── 退税申报期限内

退税计算：
├── 生产企业：免抵退
├── 外贸企业：退税=进项×退税率
└── 小规模纳税人：免税
```

### 三、进口税务要点

```
进口增值税：
├── 关税+进口增值税
├── 增值税计算：组成价格×税率
├── 消费税（如适用）
└── 跨境电商综合税（行邮税）
```

### 四、实务要点

```
跨境电商税务实务：
1. 选择合适海关模式
2. 合规进行出口退税
3. 准确归类商品编码
4. 规范保管交易凭证
5. 关注税收优惠政策
```

## 第512轮：股权激励税务专题

### 一、股权激励类型

```
常见股权激励类型：
├── 股票期权
├── 限制性股票
├── 股票增值权
├── 员工持股计划
└── 虚拟股权
```

### 二、税务处理

```
税务处理要点：
├── 授予时：一般不纳税
├── 行权时：工资薪金所得
├── 转让时：财产转让所得
└── 股息：红利所得

计算方式：
├── 行权收入 = (行权价-授予价)×股票数量
├── 税率：3%-45%超额累进
└── 股权激励单独计税
```

### 三、实务要点

```
股权激励税务筹划：
1. 激励方案设计考虑税务
2. 选择合适行权时点
3. 合理确定行权价格
4. 规划解锁安排
5. 关注递延纳税政策
```
