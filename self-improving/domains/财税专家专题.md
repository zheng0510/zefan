# 财税专家专题 - 学习笔记

## 第561轮：BEPS 2.0与全球最低税

### 一、BEPS 2.0框架

#### 1.1 双支柱核心内容

```
BEPS 2.0双支柱：
├── 支柱一（Pillar One）
│   ├── 新征税权分配（Amount A）
│   ├── 市场国享有额外征税权
│   ├── 适用范围：超大型跨国企业
│   └── 金额A计算与分配
│
└── 支柱二（Pillar Two）
    ├── 全球最低税15%
    ├── 收入纳入规则（IIR）
    ├── 低税利润规则（UTPR）
    └── 符合条件的实体排除
```

#### 1.2 支柱二核心规则

```
全球最低税实施机制：
├── 收入纳入规则（IIR）
│   ├── 母公司就子公司低税利润补征
│   └── 按比例计算
│
├── 低税利润规则（UTPR）
│   ├── 补充机制
│   └── 调整计算
│
└── 申报要求
    ├── GloBE信息申报表
    └── 各国国内法转化
```

### 二、对中国企业影响

#### 2.1 受影响企业

```
主要影响对象：
├── 超大型跨国企业（支柱一）
├── 年度合并收入≥7.5亿欧元
├── 在多个税收管辖区运营
└── 有效税率低于15%
```

#### 2.2 应对策略

```
企业应对：
├── 评估全球架构
├── 计算各辖区有效税率
├── 调整供应链和知识产权布局
├── 加强税务合规和信息披露
└── 关注国内法转化进展
```

## 第562轮：加密货币税务

### 一、加密货币税务处理

#### 1.1 税务属性认定

```
加密货币税务定性争议：
├── 货币属性
│   └── 增值税/消费税处理
│
├── 财产属性
│   ├── 财产转让所得
│   └── 资本利得税
│
├── 商品属性
│   └── 增值税处理
│
└── 中国立场
    ├── 不承认法定货币地位
    ├── 禁止相关交易（部分）
    └── 挖矿行为禁止
```

#### 1.2 国际实践

| 国家 | 税务处理 | 税率 |
|------|----------|------|
| 美国 | 财产属性，资本利得 | 0%-20% |
| 日本 | 杂项所得 | 15%-55% |
| 新加坡 | 不征收资本利得税 | - |
| 德国 | 持有1年以上免税 | - |

### 二、挖矿与Staking税务

```
税务处理难点：
├── 挖矿所得认定
│   ├── 营业收入
│   └── 资产取得成本
│
├── Staking奖励
│   ├── 收入实现时点
│   └── 公允价值计量
│
└── 中国禁止挖矿
    └── 政策风险极高
```
