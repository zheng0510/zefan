# 法律专家专题 - 学习笔记

## 第551轮：复杂跨境争议解决

### 一、跨境争议类型

#### 1.1 争议分类

```
跨境争议主要类型：
├── 国际货物买卖合同争议
│   ├── CIF/FOB等贸易术语解释
│   ├── 货物质量/数量争议
│   └── 货款支付争议
│
├── 国际技术转让争议
│   ├── 技术许可费支付
│   ├── 技术指标达标争议
│   └── 知识产权保护
│
├── 国际投资争议
│   ├── 征收补偿
│   ├── 公平待遇
│   └── 汇兑限制
│
├── 国际工程承包争议
│   ├── 工期延误
│   ├── 工程质量
│   └── 工程款支付
│
└── 国际金融交易争议
    ├── 贷款协议违约
    ├── 担保效力争议
    └── 跨境担保执行
```

#### 1.2 争议解决路径选择

| 路径 | 优势 | 劣势 | 适用场景 |
|------|------|------|----------|
| 诉讼 | 判决有强制力 | 执行难、程序长 | 有财产可执行 |
| 仲裁 | 国际执行力强 | 费用高 | 跨国交易首选 |
| 调解 | 灵活、保持合作 | 无强制力 | 希望保持关系 |
| 谈判 | 成本最低 | 易陷入僵局 | 争议初期 |

### 三、跨境争议证据问题

#### 3.1 证据收集难点

```
跨境取证难点：
├── 证据在其他法域
│   ├── 需通过司法协助途径
│   ├── Hague Evidence Convention

│   └── 形式效力认定

#### 3.2 电子证据问题

```

### 四、跨境判决与裁决执行

#### 4.1 外国判决承认与执行

```
中国《民事诉讼法》规定：
├── 外国判决承认与执行条件
│   ├── 存在条约关系或互惠关系
│   ├── 不违反中国法律基本原则
│   ├── 不损害中国主权、安全、社会公共利益
│   └── 判决已生效
│
└── 审查程序
    ├── 形式审查为主
    ├── 不审查实质性问题
    └── 听证程序
```

│   └── 不违反公共政策
│
├── 拒绝执行情形

│   ├── 仲裁协议无效

│   ├── 超裁
│   ├── 仲裁庭组成不当

│   └── 争议事项不可仲裁

└── 中国实践

    ├── 公共政策例外适用

## 第552轮：国际投资仲裁

### 一、ISDS机制概述

#### 1.1 ISDS基本概念

```
ISDS（Investor-State Dispute Settlement）：
├── 投资者 vs 东道国政府
├── 依据：双边投资协定（BIT）
├── 争端类型：征收、公平待遇、汇兑限制
└── 仲裁机制：ICSID、UNCITRAL等
```

#### 1.2 主要ISDS机构

| 机构 | 规则 | 特点 |
|------|------|------|
| ICSID | ICSID公约 | 世界银行下设，专业性强 |
| PCA | 联合国 rules | 常设仲裁法院 |
| ICC | ICC规则 | 国际商会仲裁院 |
| SCC | SCC规则 | 斯德哥尔摩商会 |

### 二、ISDS实务要点

#### 2.1 仲裁请求要点

```
仲裁请求书要素：
├── 仲裁庭管辖权
│   ├── BIT有效性
│   ├── 投资者资格
│   └── 投资定义符合
│
├── 实体权利主张
│   ├── 征收（直接/间接）
│   ├── 公平与公正待遇
│   ├── 充分保护与安全
│   └── 不歧视待遇
│
└── 赔偿请求
    ├── 全部价值赔偿
    ├── 利息计算
    └── 费用承担
```

#### 2.2 管辖权抗辩

```
东道国常用抗辩：
├── 管辖权异议
│   ├── 不属于"投资"
│   ├── 投资者不符合资格
│   └── 未满足"冷却期"
│
├── 案件实体抗辩
│   ├── 措施合法合理
│   ├── 基于公共利益
│   └── 符合BIT例外条款
│
└── 赔偿数额抗辩
    ├── 计算依据不足
    ├── 应当扣减部分
    └── 利息计算不合理
```

### 三、中国视角

#### 3.1 中国ISDS实践

```
中国ISDS发展：
├── 早年作为被诉国（有限）
├── "一带一路"投资增加，风险上升
├── 近年作为投资者母国，主动发起案件增加
└── 新一代BIT强化保护标准
```

### 四、实务要点

```
ISDS实务要点：
1. 事前：投资前评估ISDS保护水平
2. 事中：保留完整投资证据
3. 选择：仲裁机构、适用法律、仲裁员
4. 策略：管辖权争议策略、实体主张策略
5. 赔偿：经济损失量化专家证人
```
