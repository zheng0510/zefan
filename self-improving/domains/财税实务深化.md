# 财税实务深化 - 学习笔记

## 第421轮：税务稽查应对策略

### 一、税务稽查概述

#### 1.1 稽查程序流程

```
税务稽查程序
├── 选案阶段
│   ├── 税收风险分析
│   ├── 纳税评估异常
│   ├── 举报案件
│   └── 上级交办
│
├── 检查阶段
│   ├── 立案审批
│   ├── 实施检查
│   │   ├── 查账检查
│   │   ├── 实地检查
│   │   ├── 询问调查
│   │   └── 调取账簿
│   └── 检查终结
│
├── 审理阶段
│   ├── 案件审理
│   ├── 重大案件集体审理
│   └── 审理报告
│
└── 执行阶段
    ├── 税务处理决定
    ├── 税务行政处罚
    └── 强制执行（必要时）
```

#### 1.2 企业应对原则

```
应对原则：
├── 配合义务：依法接受检查、提供资料
├── 权利保护：陈述申辩、听证、复议、诉讼
├── 证据意识：保留完整财务凭证
└── 专业支持：聘请税务师、律师协助
```

### 二、常见稽查风险点

#### 2.1 增值税风险点

```
高风险行为：
├── 虚开增值税发票
├── 进项税额抵扣不合规
│   ├── 无真实业务抵扣
│   ├── 发票与业务不一致
│   └── 不得抵扣项目抵扣
├── 销项税额申报不完整
│   ├── 隐瞒收入
│   ├── 视同销售未申报
│   └── 混合销售处理错误
└── 税率适用错误
```

#### 2.2 企业所得税风险点

```
高风险行为：
├── 成本费用扣除不合规
│   ├── 无合法有效凭证
│   ├── 与经营无关支出
│   └── 超标准扣除
├── 收入确认不完整
│   ├── 账外收入
│   ├── 延迟确认收入
│   └── 关联交易定价不合理
├── 资产处理不合规
│   ├── 资产损失扣除不合规
│   └── 折旧摊销计算错误
└── 税收优惠适用错误
```

### 三、稽查应对策略

#### 3.1 检查前准备

```
自查清单：
□ 发票管理是否规范
□ 账务处理是否完整
□ 成本费用凭证是否齐全
□ 税收优惠是否符合条件
□ 关联交易定价是否合理
□ 财务人员是否熟悉业务
□ 资料归档是否完整
```

#### 3.2 检查中配合

```
注意事项：
├── 指定专人对接
├── 提供资料清单签收
├── 重要问题咨询专业顾问
├── 记录检查人员提问重点
├── 发现问题及时说明解释
└── 保持沟通态度良好
```

#### 3.3 争议处理

```
争议解决路径：
异议 → 陈述申辩 → 听证 → 行政复议 → 行政诉讼

关键要点：
├── 收到处理决定前提出陈述申辩
├── 重大案件申请听证
├── 对决定不服可在60日内申请复议
├── 复议前置：税款争议须先缴纳税款
└── 对复议不服可在15日内提起诉讼
```

### 四、实务要点总结

1. **事前防范优于事后补救**：日常合规最重要
2. **配合义务与权利保护并重**：既要配合也要维权
3. **证据是核心**：完整凭证是抗辩基础
4. **专业支持很关键**：复杂案件请专业人员协助
5. **程序要合法**：稽查程序违法可成为抗辩理由

## 第422轮：转让定价实务

### 一、转让定价概念

#### 1.1 什么是转让定价

**定义：** 关联企业之间交易价格的确定方式

```
适用场景：
├── 母子公司之间货物交易
├── 关联企业之间提供服务
├── 集团内部资金融通
└── 无形资产许可使用
```

#### 1.2 独立交易原则

```
核心原则：关联企业之间的交易价格
          应当与非关联企业在相同条件下交易价格一致

验证方法：
├── 可比非受控价格法
├── 再销售价格法
├── 成本加成法
├── 交易净利润法
└── 利润分割法
```

### 二、关联交易类型与风险

#### 2.1 常见关联交易

| 类型 | 典型情形 | 税务风险 |
|------|----------|----------|
| 有形资产交易 | 货物购销 | 价格偏离公允价格 |
| 无形资产交易 | 专利许可使用 | 特许权使用费过高/过低 |
| 劳务交易 | 管理服务 | 服务费不合理 |
| 融资交易 | 资金借贷 | 利率偏离市场水平 |

#### 2.2 税务风险分析

```
转让定价风险：
├── 利润转移嫌疑
│   └── 利润从高税区转移至低税区
├── 税基侵蚀风险
│   └── 过高成本减少应税所得
├── 预约定价安排未申请
└── 同期资料未准备
```

### 三、转让定价方法详解

#### 3.1 可比非受控价格法（CUP）

```
适用条件：
├── 存在可比非关联交易
└── 交易条件高度相似

应用步骤：
├── 确定非关联交易价格
├── 调整差异因素
└── 确定关联交易合理价格
```

#### 3.2 成本加成法

```
适用条件：
├── 适用于半成品、劳务交易
└── 成本可准确计算

计算公式：
合理价格 = 成本 × (1 + 合理加成率)

加成率确定：
└── 参照同行业、同类型企业的加成率
```

#### 3.3 交易净利润法（TNMM）

```
适用条件：
├── 交易双方功能风险相似
└── 难以找到可比价格

核心指标：
├── 净利润率
├── 资产收益率
└── 完全成本加成率

比较基准：
└── 可比企业相同指标的四分位区间
```

### 四、同期资料准备

#### 4.1 准备要求

```
准备门槛：
├── 年度关联购销金额超过2亿元
└── 其他关联交易金额超过4000万元

资料内容：
├── 组织结构
├── 生产经营情况
├── 关联交易情况
├── 可比性分析
└── 转让定价方法选择
```

### 五、预约定价安排

#### 5.1 APA类型

```
┌─────────────────────────────────────┐
│ 单边APA：企业与一国税务机关签订    │
│ 双边APA：企业与两国税务机关签订    │
│ 多边APA：涉及三方及以上税务机关    │
└─────────────────────────────────────┘
```

#### 5.2 APA流程

```
预备会谈 → 正式申请 → 谈判 → 签署 → 执行
```

### 六、实务要点总结

1. **独立交易原则是核心**：所有方法均以此为基础
2. **功能风险分析是前提**：明确各方的功能和风险承担
3. **可比性分析是关键**：选择可比对象需谨慎
4. **同期资料是保障**：按规定准备并保存
5. **预约定价是优选**：复杂交易建议申请APA

## 第423轮：国际税收筹划

### 一、国际税收基本概念

#### 1.1 税收管辖权

```
类型：
├── 居民管辖权：对本国居民全球所得征税
└── 来源地管辖权：对来源于本国所得征税

双重征税避免：
└── 税收协定 + 国内法消除机制
```

#### 1.2 税收协定网络

```
中国已签订协定：100+个国家/地区

核心条款：
├── 常设机构认定
├── 营业利润征税权划分
├── 股息、利息、特许权使用费限制税率
└── 消除双重征税方法
```

### 二、常见筹划架构

#### 2.1 控股公司架构

```
架构示例：
中国母公司
     ↓
香港控股公司（区域性控股）
     ↓
新加坡/荷兰中间控股
     ↓
目标投资国运营公司

筹划要点：
├── 利用税收协定降低预提税
├── 利润留存于低税区
└── 退出时转让架构降低税负
```

#### 2.2 IP持有架构

```
知识产权公司设立考虑：
├── 研发费用加计扣除（中国）
├── IP公司所在国税率
├── 特许权使用费预提税
└── "税基侵蚀"规则影响
```

### 三、国际反避税规则

#### 3.1 BEPS行动计划

```
BEPS 15项行动计划：
├── 行动1：数字经济税收挑战
├── 行动2：消除混合错配安排
├── 行动3：受控外国公司规则
├── 行动4：利息扣除限制
├── 行动5：有害税收实践
├── 行动6：协定滥用防止
├── 行动7：常设机构定义修改
├── 行动8-10：转让定价
├── 行动11：数据收集分析
├── 行动12：强制披露规则
├── 行动13：国别报告
├── 行动14：争议解决机制
└── 行动15：多边公约
```

#### 3.2 反避税条款

```
主要反避税规则：
├── 受控外国企业规则（CFC）
│   └── 低税区留存利润视同分配
├── 一般反避税规则（GAAR）
│   └── 缺乏合理商业目的安排可调整
└── 主要目的测试（PPT）
    └── 主要目的为获取税收利益的安排可拒绝
```

### 四、实务要点总结

1. **商业目的优先**：商业实质比税务筹划更重要
2. **合规底线**：严格遵守各国税法规定
3. **BEPS影响深远**：传统筹划空间大幅压缩
4. **信息披露增加**：国别报告等透明度要求提高
5. **争议解决重要**：预约定价、相互协商程序

## 第424轮：并购重组税务规划

### 一、并购重组税务架构

#### 1.1 交易结构类型

| 结构 | 税务影响 | 适用场景 |
|------|----------|----------|
| 股权收购 | 原股东缴税，标的公司税务属性保留 | 标的公司有重要牌照/资质 |
| 资产收购 | 标的公司缴税，收购方获折旧基础 | 标的公司有税务风险 |
| 合并/分立 | 可能适用特殊性税务处理 | 集团内部重组 |

#### 1.2 税务尽职调查要点

```
税务尽调重点：
├── 纳税合规情况
│   ├── 税款申报是否完整
│   ├── 税收优惠是否符合条件
│   └── 是否存在欠税/稽查风险
├── 递延所得税资产/负债
├── 关联交易风险
├── 资产计税基础
└── 历史遗留税务问题
```

### 二、特殊性税务处理

#### 2.1 适用条件

```
《企业所得税》特殊性税务处理条件：
├── 具有合理商业目的
├── 股权支付比例≥85%（部分交易）
├── 原主要股东12个月内不转让股权
├── 经营连续性（12个月内不改变实质经营）
└── 原有经营活动保持连续
```

#### 2.2 具体处理方式

```
特殊性税务处理效果：
├── 暂不确认所得/损失
├── 计税基础保持不变
└── 原企业税务属性延续

vs 一般性税务处理：
├── 即时确认所得/损失
├── 计税基础按公允价值确认
└── 原企业税务属性可能丧失
```

### 三、实务要点

```
并购税务规划重点：
1. 交易结构决定税务成本
2. 尽调中发现问题可作为谈判筹码
3. 合理设计支付方式降低税负
4. 特殊性税务处理需严格满足条件
5. 交易文件中明确税务责任分配
```

## 第425轮：房地产税务实务

### 一、房地产税种体系

```
开发阶段：
├── 契税（土地取得）
├── 耕地占用税（占用耕地）
└── 城镇土地使用税

销售阶段：
├── 增值税及附加
├── 土地增值税
├── 企业所得税
└── 印花税

持有阶段：
├── 房产税
└── 城镇土地使用税
```

### 二、土地增值税

#### 2.1 计算方法

```
土地增值税 = 增值额 × 税率 - 扣除项目 × 速算扣除系数

增值额 = 收入 - 扣除项目金额

扣除项目：
├── 取得土地使用权支付金额
├── 房地产开发成本
├── 房地产开发费用（利息+其他）
├── 与转让房地产有关的税金
└── 财政部规定的其他扣除（20%加计扣除）
```

#### 2.2 税率表

| 增值率 | 税率 | 速算扣除系数 |
|--------|------|--------------|
| ≤50% | 30% | 0% |
| 50%-100% | 40% | 5% |
| 100%-200% | 50% | 15% |
| >200% | 60% | 35% |

### 三、实务要点

```
房地产税务筹划要点：
1. 成本核算准确（影响扣除金额）
2. 分期开发合理分摊成本
3. 普通标准住宅增值率控制20%以下免征
4. 清算时点选择优化
5. 合理利用税收优惠政策
```

## 第426轮：金融税务实务

### 一、金融机构税种

```
金融机构主要税种：
├── 增值税
│   ├── 贷款服务（6%）
│   ├── 直接收费金融服务（6%）
│   └── 金融商品转让（6%）
├── 企业所得税
└── 印花税
```

### 二、增值税要点

#### 2.1 贷款服务

```
贷款服务增值税：
├── 税率：6%
├── 计税依据：利息收入
├── 不得抵扣进项税额
└── 包括：利息、利息性质的收入、保本收益
```

#### 2.2 金融商品转让

```
金融商品转让增值税：
├── 差额征税：卖出价 - 买入价
├── 正负差可结转（年内）
├── 年末仍有负差的，不得结转下年
└── 不得开具增值税专用发票
```

### 三、实务要点

```
金融税务要点：
1. 准确划分收入类型适用税率
2. 进项税额抵扣管理
3. 金融商品转让差额计算
4. 税收优惠政策运用
5. 监管合规与税务合规衔接
```

### 二、争议解决途径

